Key Capabilities

Everything you need for enterprise networking

From development to production — all integrated, all managed.

Multi-Tenant Isolation
One namespace per subscription with overlapping CIDR support via Cilium namespace boundaries.
Multi-Cluster Resilience
3-cluster architecture (Storage, Data, Compute) with automatic failover and ClusterMesh synchronization.
Direct VLAN IP Assignment
Services get routable IPs from your network pools via BGP. External access without NAT.
Zero-Trust Security
Cilium network policies, NSG rules, mTLS, and audit logging by default.

Features

Enterprise-grade networking

Purpose-built for multi-tenant Kubernetes at scale.

Virtual Networks
Create and manage isolated networks with configurable address spaces.
Network Security
NSGs, Cilium policies, mTLS, and zero-trust security defaults.
Load Balancing
Layer 4 and Layer 7 load balancing with automatic VLAN IP assignment.
Private Link
Cross-subscription private endpoints with DNS integration.
BGP Routing
Direct VLAN IP advertisement to physical network. No NAT required.
Observability
Prometheus metrics, Grafana dashboards, and audit logging included.
Multi-Tenant Security
Strict namespace isolation and overlapping CIDR support per tenant.
IP Address Management
Automatic IP allocation with pool management and capacity planning.
